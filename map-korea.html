Kakao Map

Usually in Korea, highly recommend you to use local application like kakao map or naver map because sometimes google map catch your GPS location wrongly.
This application provides information like bus lane, bus stop, metro and recommended restaurant etc. based on your location or your search. You can check the fastest way to get to your destination with lots of options and the fare when use public transport. Through this app, you can see the map with 360 degrees or sky view. Many users evaluate this app is easy to use and accurate than other apps.

Naver Map

This application also provide same features of kakao map. This app has a feature of download, so you can use it without the internet. But there are some err about bus lane or bus stop. Still lots of people complement because of a feature of finding location. You can call a taxi using this app. (kakao map also has a same feature)

Seoul Metro Map

This website provide almost all about information about metro facilities like wheelchair lift or breast-feeding room etc. You can easily check these kinds of information through below area of the website. If you want to visit the website just click the left logo. Want to know how to purchase and use a metro ticket? The basic fare is 1,250 won.
